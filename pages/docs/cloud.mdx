We're excited to share our Cloud offering! This guide covers everything you need to get integrated
using your own repository with Grouparoo Cloud.

### Step 1 - Register a Cloud Account

To make a Cloud account, visit [Cloud](https://cloud.grouparoo.com/) and press Register.

<Image
  src="docs/cloud/cloud-register.png"
  alt="Cloud App Register"
  width={1802}
  height={936}
/>

Register will take you to the [Trial Request](/trial) page. Our Administrators will help get you set up in our system and ready to integrate! Once your account is created, revisit [Cloud](https://cloud.grouparoo.com/) and sign in.

<Image
  src="docs/cloud/signin.png"
  alt="Cloud App Sign In"
  width={1784}
  height={426}
/>

### Step 2 - Integrating with Grouparoo Cloud

Upon logging in you should be presented with a fresh project screen.

<Image
  src="docs/cloud/project-landing-page.png"
  alt="Cloud App Fresh Project Landing Page"
  width={1544}
  height={469}
/>

Grouparo Cloud is designed to work with _your_ GitHub repository! To set up a Code Config repository, please follow our [Docs](/docs/getting-started) on how to do so.

Once your repository is setup and pushed up to Github, go [here](/docs/cloud/repository-integration).
