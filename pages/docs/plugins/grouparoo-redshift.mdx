Grouparoo's [Redshift](https://aws.amazon.com/redshift/) plugin enables you to import data from one or more Redshift databases or to export data to a Redshift database.

This guide will show you how to work with the Redshift plugin to [create a Source](#create-redshift-source) to import your data, or to [create a Destination](#create-redshift-destination) to export your data.
