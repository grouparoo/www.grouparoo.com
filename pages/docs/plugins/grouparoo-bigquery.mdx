Grouparoo's [BigQuery](https://cloud.google.com/bigquery/) plugin enables you to import data from one or more BigQuery databases or to export data to a BigQuery database.

This guide will show you how to work with the BigQuery plugin to [create a Source](#create-bigquery-source) to import your data, or to [create a Destination](#create-bigquery-destination) to export your data.
