Grouparoo's [MySQL](https://www.mysql.com/) Plugin enables you to import data from one or more MySQL databases or to export data to a MySQL database.

This guide will show you how to work with the MySQL Plugin to [create a Source](#create-mysql-source) to import your data, or to [create a Destination](#create-mysql-destination) to export your data.
