Grouparoo can be extended with [Plugins](/docs/plugins) to add more Apps, Sources, and Destinations.

<Button href="/docs/plugins">Learn more about plugins</Button>

<br />
<br />

The Grouparoo command line tool is used to add a Plugin to your project via `grouparoo install [package]`. For example:

```bash
grouparoo install @grouparoo/mysql # install the latest version
```

You can learn more about the `grouparoo install` command [here](/docs/cli/setup).
