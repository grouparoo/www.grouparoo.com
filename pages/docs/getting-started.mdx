Welcome!

We're so glad you've chosen to spend some time with Grouparoo. This guide will help you get started by running Grouparoo locally on your computer or in a trial environment.

## Installation

There are three ways to quickly install Grouparoo. For more options and for information on deploying to a production environment, see [the Installation & Deployment section](/docs/deployment).

<h3>Run locally with Docker & Docker Compose</h3>

If you have [Docker](https://www.docker.com/) installed, you can also run Grouparoo locally:

```
# in a new directory
curl -L https://www.grouparoo.com/docker-compose --output docker-compose.yml
docker-compose up
```

  </Tab>

<h3>Run Locally with Node.js</h3>

You can also try Grouparoo locally with Node.js:

```
# in a new directory
npx grouparoo init .
npm start
```
