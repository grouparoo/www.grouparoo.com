Welcome! We're so glad you've chosen to spend some time with Grouparoo. See below for getting started with the basics.

## Installation

There are two ways to quickly install Grouparoo. For more options and for information on deploying to a production environment, see [the Installation & Deployment section](/docs/deployment).

### Deploy to Heroku

The simplest way to see Grouparoo in action is to deploy it to Heroku for free:

[![Deploy to Heroku](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy?template=https://github.com/grouparoo/app-example)

### Run Locally

You can also try Grouparoo locally:

    # in a new directory
    npx grouparoo init .
    npm start
