The 0.8 release had a big focus on user experience with the addition of our Model-Centric UI!

<Image
  alt="Salesforce Destinations"
  src="v08-release/model-centric-ui.png"
  width={1000}
  height={424}
/>

We now support sending metrics to Rollbar.

And added a few more integrations:

- Airtable
- Klaviyo
- Vero

And have improved how we interact with Google Sheets as a Source!

There are some breaking changes in this release so please see [this](https://www.grouparoo.com/docs/support/upgrading-grouparoo/v07-v08#migration-notes-and-breaking-changes) page for details and instructions on how to upgrade.
